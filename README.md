\## 📊 Metodología de Comunicación: Estándar C➔F➔I

Cada proyecto en este portafolio entrega resultados bajo el framework \*\*Contexto ➔ Hallazgo ➔ Implicación\*\*, diseñado para ser interpretado por directivos en menos de 5 segundos \[5, 18, 19]:



\*   \*\*Contexto (C):\*\* Delimitación clara del problema o KPI evaluado.

\*   \*\*Hallazgo (F):\*\* Declaración de la cifra clave con unidad de medida y comparativo explícito.

\*   \*\*Implicación (I):\*\* Recomendación de negocio accionable con impacto financiero estimado.

\## 🛡️ Gobernanza de Datos y Buenas Prácticas

Este repositorio sigue estándares de \*\*Ingeniería Senior\*\* \[20, 21]:

1\.  \*\*Arquitectura Limpia:\*\* Separación estricta de `/data/raw`, `/src/`, `/sql/` y `/notebooks/` \[22, 23].

3\.  \*\*Código de Producción:\*\* Migración de entornos experimentales (.ipynb) a scripts modulares (.py) legibles (PEP 8) \[9, 10, 26].
