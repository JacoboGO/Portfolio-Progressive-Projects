# 📂 Proyecto 1 — Fundamentos de Negocio y Velocidad Operativa

## Resumen

Auditoría de aproximadamente **250,000 transacciones** para evaluar desempeño logístico y comercial.

## Objetivos

* Validar calidad de datos.
* Analizar Lead Time.
* Identificar outliers mediante IQR.
* Evaluar Revenue y Margen Bruto.
* Generar recomendaciones para negocio.

## Stack Tecnológico

|Tecnología|Uso|
|-|-|
|Excel 2013|Limpieza y modelado|
|Google Sheets|Automatización|
|Git|Versionamiento|
|Git Bash|Gestión del repositorio|

## Arquitectura de Datos

```text
Datos → Limpieza → Modelado Estrella → KPIs → Insights → Recomendaciones
```

## Principales Hallazgos

### Lead Time

* Distribución sesgada a la derecha.
* Mediana recomendada como KPI principal.
* Concentración de retrasos en Bodega A.

### Rentabilidad

* Revenue: **$452.1M**
* GPM: **42.7%**
* Dependencia comercial de Bodegas A y B.

## Recomendaciones

* Auditar procesos de despacho.
* Redistribuir inventario.
* Priorizar métricas robustas frente a valores extremos.
